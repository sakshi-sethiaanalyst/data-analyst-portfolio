# SLA & Operational Performance Dashboard
**Tools:** Power BI | DAX | SQL

## Overview
This dashboard monitors SLA compliance and operational performance 
for a simulated financial services dataset of 1,000+ case records. 
Built to reflect real-world reporting used in regulated banking and 
insurance environments.

## Business Problem
Operations teams in regulated environments need real-time visibility 
into SLA breaches, case ageing, and resolution bottlenecks to maintain 
compliance and reduce operational risk.

## What I Built
- End-to-end Power BI dashboard tracking SLA breach rates, case 
  ageing profiles, and resolution timelines
- DAX measures to calculate breach %, average handling time, and 
  workload distribution
- SQL queries to clean, transform, and load the simulated dataset

## Key Insights
- Identified top delay drivers contributing to 80% of SLA breaches
- Visualised ageing profiles by case type to support prioritisation
- Built risk indicators to flag high-exposure cases for escalation
